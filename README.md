# Repositorio de Música Libre / Free Music Repository

![1778080987540](images/README/1778080987540.png)

## Descripción / Description

Este repositorio contiene una colección organizada de carpetas y archivos Markdown que documentan música libre de derechos, música sin copyright y canciones para creadores. Cada carpeta representa un canal, artista o compilación, y dentro de ellas hay archivos `.md` con los detalles de las pistas, títulos y artistas.

This repository contains an organized collection of folders and Markdown files documenting royalty-free music, no-copyright tracks, and songs for creators. Each folder represents a channel, artist, or compilation, and inside them are `.md` files with track details, titles, and artists.

## Utilidad / Purpose

- Organizar y compartir música gratis para uso en videos, proyectos y contenidos.
- Facilitar la búsqueda de pistas libres de derechos en categorías y canales conocidos.
- Servir como referencia para creadores que necesitan encontrar música segura para monetización.
- Organize and share free music for use in videos, projects, and content.
- Make it easier to find copyright-free tracks by category and known channels.
- Serve as a reference for creators looking for music safe for monetization.

## Estructura / Structure

- Cada carpeta corresponde a un canal, artista o etiqueta musical.
- Los archivos dentro de cada carpeta son notas o listados de canciones, generalmente en formato Markdown.
- No contiene los archivos de audio directamente, sino más bien referencias y descripciones de música disponible.
- Each folder corresponds to a channel, artist, or music label.
- The files inside each folder are notes or song listings, typically in Markdown format.
- It does not contain the audio files directly, but rather references and descriptions of available music.

## Cómo usarlo / How to use it

1. Explora las carpetas por artista o canal.
2. Abre los archivos `.md` para ver los títulos y la información de cada pista.
3. Usa la estructura de carpetas para encontrar rápidamente música libre de derechos según el estilo o la fuente.
4. Browse folders by artist or channel.
5. Open the `.md` files to see titles and information for each track.
6. Use the folder structure to quickly find copyright-free music by style or source.

## Nota / Note

Este repositorio es una colección de referencias. Asegúrate siempre de verificar las licencias y condiciones de uso específicas de cada pista antes de utilizarla en proyectos comerciales o públicos.

This repository is a reference collection. Always verify the specific license and usage conditions of each track before using it in commercial or public projects.
